# Pepper Challenge : Groupe 17 - Takeo
**Auteurs :** *Arnaud Saison, Michaël Feldman, Hind Bakkali Tahiri, Younes Bouhjar, Idil Ari, Joshua Nicdao Blanco*

Le logiciel a été créé sur base du contenu du rapport. Merci de s'y référer pour toute justification apporfondie.
## Utilisation du logiciel
- Exécutez le fichier *logicielgroupe17.py*
- Vous pouvez exécutez le code dans n'importe quelle console python 3.
- Le programme nécessite [l'installation des paquets scipy](https://www.scipy.org/install.html)
- Le programme dispose de quatre modes de fonctionnemnet détaillés lors du lancement de celui-ci, veillez à tuiliser le bon.
- Les couleurs n'ont pour seule utilité que de mettre en évidence les résultats importants et les rendre plus lisibles.
- Les résultats sont arrondis à un nombre de décimales dépendant de la précision requise.
## Troubleshooting
Si le code ne semble pas fonctionner correctement :
- Vérifier que les modules requis sont installés.
- Il faut garder à l'esprit que le logiciel a été conçu pour un certain domaine de validité, qui est le séchage du poivre de Kampot au Cambodge. 
Il n'est pas pas impossible que dans des conditions très différenes, 
d'autres phénomènes physiques interviennent et que des erreurs apparaissent lors de l'exécution.
L'indicateur de validité des corrélations (vérifié sur base du nombre de Rayleigh) donne une première idée de si l'on se trouve dans le domaine de validité.
- Un message d'erreur peut occasionnellement apparaître suite à la résolution des équations. 
Celui-ci n'influence pas les résultats.
